# AskFirst Task Manager

A simple and responsive Task Manager application built using Next.js and Zustand for state management.  
The application allows users to create, manage, filter, and persist tasks locally.

## 🚀 Features

- Add new tasks with title and priority level
- Set task priority:
  - Low
  - Medium
  - High
- View all created tasks
- Mark tasks as completed or pending
- Filter tasks based on status:
  - All Tasks
  - Pending Tasks
  - Completed Tasks
- Persistent storage using localStorage
- Clean and responsive user interface


## 🛠️ Tech Stack

- Next.js
- React.js
- Zustand (State Management)
- Zustand Persist Middleware
- Tailwind CSS
- localStorage
